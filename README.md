### ⚡️About ddip
ddip은 핵 개인화된 사회의 니즈에 따라 사람들이 부담 없이 취미활동을 즐길 수 있도록 돕기 위해 만들었습니다.

복잡한 관계나 소속은 부담스럽지만 당장 오늘 저녁, 내일, 이번 주 주말 같이 놀 사람이 필요하신 분들이 이용하기에 적합한 웹 애플리케이션입니다.
